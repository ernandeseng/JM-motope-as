# **App Name**: JM Moto Peças Atacarejo Landing Page

## Core Features:

- Hero Section: Display a hero section with a high-quality motorcycle image, logo, headline, subheadline, and a prominent call-to-action button.
- Featured Products Section: Showcase a grid of featured products with images, titles, and 'Falar com um Vendedor' buttons.
- Brands Carousel: Implement an automatically scrolling carousel featuring logos of major motorcycle brands.
- Unique Selling Points: Present cards with icons and text highlighting the unique benefits of shopping at JM Moto Peças, such as original parts, fast delivery, wholesale prices, and specialized support.
- Customer Testimonials: Showcase customer testimonials with photos, names, text, and star ratings.
- WhatsApp CTA Section: Create a dedicated section with a WhatsApp icon, title, subtitle, and a call-to-action button linking to the WhatsApp contact.
- Footer Section: Develop a footer with the JM Moto Peças logo, a brief description, quick links, contact information, social media icons, and a copyright notice.

## Style Guidelines:

- Primary color: Vibrant red (#E63946), extracted from the logo, will be used for primary buttons and accents to capture attention and reflect the brand's energy.
- Secondary color: Yellow/Gold (#FFD60A), also extracted from the logo, will highlight key interactive elements like 'Falar com um Vendedor' buttons to add a touch of premium feel and guide user actions.
- Background: Use white (#FFFFFF) for main sections to ensure readability and cleanliness, alternating with light gray (#F5F5F5) for sections to create visual separation.
- Text/Contrast: Dark gray (#1A1A1A) for text and contrast elements to provide excellent readability against light backgrounds.
- Headline Font: 'Montserrat' (sans-serif) for titles to convey modernity and boldness.
- Body Font: 'Open Sans' (sans-serif) for body text to ensure readability and a clean aesthetic.
- Employ icons from libraries such as Font Awesome or Lucide to represent various concepts, using yellow/gold (#FFD60A) for visual consistency.
- Modern, clean, and professional layout, fully responsive, using HTML5, CSS3 with Flexbox/Grid.
- Incorporate subtle animations, such as fade-in and slide-up effects, during scrolling for enhanced user experience.
